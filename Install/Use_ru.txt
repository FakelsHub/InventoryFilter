Использование:
Используйте среднюю кнопку мыши на кнопках меню "Оружие" и "Патроны" для одновременного отображения предметов этих категории в инвентаре игрока,
так же клик средней мышки клавишей по кнопке "Разное" позволяет вам отобразить полезные предметы из категории "Разное".

Для быстрого переключения между категориями фильтра используйте горячие клавиши, для:
Игрока: Q, 1, 2, 3, 4, 5
НПЦ:    W, 6, 7, 8, 9, 0
Изменить горячие клавиши вы можете в конфигурационном файле.

Для включения или выключения плагина в случае проблем в игре с фильтром используйте клавишу F11.

При обмене с контейнерами или партийцами вместо фильтрации предметов по их типу, используется сортировка предметов для выбранной категории.
