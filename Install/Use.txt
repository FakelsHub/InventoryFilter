Usage:
Use the middle mouse button on the "Weapons" and "Ammo" menu buttons to simultaneously display items of these categories in the player's inventory,
also clicking the middle mouse button on the "Weapons" button allows you to display only guns weapon.

To quickly switch between filter categories, use the hot-keys:
Player: Q, 1, 2, 3, 4, 5
NPC:    W, 6, 7, 8, 9, 0
Change keyboard hot-keys, you can in the configuration file.

For activate or deactivate the filter use the F11 key.
