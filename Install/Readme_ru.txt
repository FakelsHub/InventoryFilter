Inventory Filter version 2.0 by Mr.STALIN
 
Плагин работоспособен с новыми версиями начиная с sFall v4.2.8.1
ВНИМАНИЕ: Для корректной работы фильтра требуется включение опции AllowUnsafeScripting в конфигурационном файле ddraw.ini (включается автоматически при установке).

Дополнительные возможности:
Режим Smart Barter - позволяет игроку во время торговли не заботиться о денежных средствах по окончанию бартерной сделки. 
Режим Swap - позволяет удобно переключаться между напарниками во время торговли или обмена с контейнерами/багажником.
Добавляет кнопку "Положить все" для интерфейса обмена с контейнерами и напарниками.
