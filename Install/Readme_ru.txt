Inventory Filter version 1.6.1 by Mr.Stalin
 
Плагин полностью работоспособен с версиями sFall v4.2.2+ с более ранними версиями возможны некоторые ошибки в его работе.
Внимание: Для правильной работы фильтра потребуется включение опции AllowUnsafeScripting в конфигурационном файле ddraw.ini.

Нововведения:
К полезным вещам категории "Разное", для отображения добавлены некоторые предметы из Olympus2207 и Fallout 2: Nevada. (предложение от Foxx)
Режим Smart Barter - позволяющий игроку во время торговли не заботиться о денежных средствах по окончанию бартерной сделки. 
Режима Swap - для удобного переключения между напарниками во время торговли или обмена с машиной.
Новая кнопка "Положить Все" доступна только для sFall v4.1.0+ с включенной опцией InterfaceDontMoveOnTop
Назначение горячих клавиш для переключения между режимами фильтрации (для sFall v4.2 и выше).
